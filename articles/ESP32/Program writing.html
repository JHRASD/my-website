一· 程序的结构

一个单片机的程序主体结构，由头文件，setup和loop组成，setup是单片机只执行一次的函数，loop是单片机会一直循环执行的函数

#include 

void setup() {
// 初始化代码（只执行一次）
}

void loop() {
// 循环执行的代码（一直重复）
}

setup()：开机时执行一次
loop()：不停循环执行（每秒执行很多次）

二、GPIO（数字输入输出）

1. 什么是 GPIO？

GPIO 通用输入输出引脚 也就是开发板上面的引脚，每个针脚都可以输出电压(3.3v或0v)，读取电压 用来接按钮，传感器，屏幕等...常用的引脚

2.GPIO有四种模式

  • 输出 OUTPUT
  • 输入 INPUT
  • 高电平 HIGH
  • 低电平 LOWT

GPIO常用的函数：

1. pinMode(pin,mode);//设置引脚是输入还是输出

例:

  • pinMode(2,OUTPUT); 设置针脚2为输出模式
  • pinMode(4,INPUT); 设置针脚4为输入模式
  • pinMode(4,INPUT_PULLUP); 设置针脚4为输入模式,并带内部的上拉电阻

2. digitalWrite(pin,value); //设定针脚的输出是高或低

例:

  • digitalWrite(2,HIGH); 设置针脚2为高电平
  • digitalWrite(4,LOW); 设置针脚4为低电平

3. digitalRead(pin); //读取引脚的电平

例:

  • digitalRead(4); //读取针脚4的电平，返回HIGH或LOW
